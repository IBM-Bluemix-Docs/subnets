# 글로벌 IP 주소 정보
{:#about-global-ip-address}

글로벌 IP 주소는 특수화된 정적 보조 서브넷입니다. 계정의 다른 IP 주소로 라우팅할 수 있는 /32 서브넷(즉, 단일 IP 주소)으로 전달됩니다. IPv4와 IPv6 주소를 둘 다 사용할 수 있으며 각 유형은 동일한 IP 버전의 IP 주소로 라우팅해야 합니다. 승인 가능 라우팅 대상에는 서버에서 사용 중인 기본 IP 주소와 휴대용 보조 서브넷 IP 주소가 포함됩니다. 글로벌 IP 주소의 고유 기능은 다음과 같습니다.

  * 요구 시 계정의 IP 주소에 글로벌하게 라우팅합니다.
  * 주소는 모든 {{site.data.keyword.cloud}} 에지 라우터에서 인터넷에 공지합니다. 그러므로 데이터는 {{site.data.keyword.cloud}} 네트워크의 최단 경로를 선택하고 여기에서 {{site.data.keyword.cloud}} 데디케이티드, 글로벌 백본을 사용하여 구성한 대상에 연결할 수 있습니다. 

글로벌 IP 주소는 유연성을 제공합니다. 이 주소를 통해 지리적으로 서로 다른 데이터 센터에 있는 서버 간에도 워크로드를 이동할 수 있습니다. 또한 글로벌 IP 주소를 사용하면 DNS 캐시 방지 등을 위해 조정할 필요 없이 상태 전이를 허용하여 IP 지속성을 제공합니다. 글로벌 라우팅 기능은 재해 복구 사이트 간에 워크로드를 전이하거나 대상에게 서비스 제공을 개선할 수 있는 지역의 새 배치로 원활하게 전이하는 데 적합합니다.

## 글로벌 IP 주소 관리
{:#manage-global-ip-address}

글로벌 IP 주소를 관리하려면 다음 단계를 따르십시오.

 1. 브라우저에서 [고객 포털 ![외부 링크 아이콘](../../icons/launch-glyph.svg "외부 링크 아이콘")](https://{DomainName}/){: new_window}을 열고 사용자 계정에 로그인하십시오.
 1. 고객 포털 탐색에서 **클래식 인프라**를 선택하십시오.
 1. 클래식 인프라 탐색 메뉴에서 **네트워크 > IP 관리> 글로벌 IP**를 선택하십시오.
 1. 글로벌 IP 주소가 나열되는 것을 확인할 수 있습니다. 필터를 활용하여 필요한 대로 검색 범위를 좁히십시오. 
 
일부 항목에는 **대상** 값이 없음을 알 수 있습니다. 즉, 글로벌 IP 주소가 현재 라우팅되지 않으므로 서비스 중이 아님을 나타냅니다.

### 주소 라우팅 및 라우팅 해제
{:#route-unroute-address}

원하는 글로벌 IP 주소를 찾고 나면 서브넷 ID를 클릭하십시오. 해당되는 경우 현재 라우트 대상을 표시하는 화면이 다음에 나타납니다. 새 대상으로 라우팅(트래픽 보내기)하려면 다음 두 옵션이 있습니다.

 * 전체 IP 주소 입력 또는
 * 호스트 이름 입력 시작.
 
호스트 이름을 입력하면 서버의 호스트 이름을 검색할 수 있으므로 해당 IP 주소를 조회할 수 있습니다. IP 주소를 입력한 다음 **업데이트**를 선택하십시오. 글로벌 IP 주소의 라우팅을 취소하려면 **지우기**를 선택하십시오. 입력 상태가 '라우팅 해제됨'으로 변경되면 **업데이트**를 선택하십시오.

드롭 다운 메뉴는 사용 가능한 IP 주소의 포괄적 목록이 아닙니다. 선택 목록에 없으면 원하는 IP 주소를 직접 입력하십시오.
{:note}

### 업데이트 빈도는 어떻게 됩니까?
{:#how-often-can-i-update}

언제든 하나의 단일 라우트 업데이트만 진행할 수 있습니다. 다른 업데이트를 진행하는 동안 글로벌 IP 주소 라우트를 업데이트하려고 하면 오류가 표시됩니다. 다시 시도하기 전에 이전 업데이트가 완료될 때까지 기다리십시오.


## 글로벌 IP 주소 주문 방법
{:#how-to-order-global-ip-address}

다음 지시사항을 따라 글로벌 IP 주소를 주문하십시오.

  1. 브라우저에서 [고객 포털 ![외부 링크 아이콘](../../icons/launch-glyph.svg "외부 링크 아이콘")](https://{DomainName}/){: new_window}을 열고 사용자 계정에 로그인하십시오.
  1. 고객 포털 탐색에서 **클래식 인프라**를 선택하십시오.
  1. 클래식 인프라 탐색 메뉴에서 **네트워크 > IP 관리> 글로벌 IP**를 선택하십시오.
  3. **새 IP 주소** 링크를 클릭하십시오.
  4. 드롭 다운 메뉴에서 필요에 따라 **글로벌 IPv4** 또는 **글로벌 IPv6**를 선택하고 **계속**을 클릭하여 주문 프로세스를 시작하십시오.

### 다음 작업
{:#global-ip-address-next}

계정 상태의 필수 승인 프로세스를 금지하면 잠시 후에 새 글로벌 IP 주소 서브넷이 계정에 표시됩니다.

### 리소스 한계
{:#global-ip-resource-limit}

계정에는 IP 버전당 5개의 글로벌 IP 주소만 포함될 수 있습니다. 예를 들어, 5개의 IPv4 글로벌 IP 주소와 5개의 IPv6 글로벌 IP 주소입니다.

## 글로벌 IP 주소 취소 방법
{:#how-to-cancel-global-ip-address}

글로벌 IP 주소가 더 이상 필요하지 않으면 다음 단계를 따라 취소하십시오.

  1. 브라우저에서 [고객 포털 ![외부 링크 아이콘](../../icons/launch-glyph.svg "외부 링크 아이콘")](https://{DomainName}/){: new_window}을 열고 사용자 계정에 로그인하십시오.
  1. 고객 포털 탐색에서 **클래식 인프라**를 선택하십시오.
  1. 클래식 인프라 탐색 메뉴에서 **네트워크 > IP 관리> 글로벌 IP**를 선택하십시오.
  1. 원하는 주소를 찾도록 필요한 필터를 완료하십시오.
  1. 주소 목록의 주소 항목 오른쪽에 X가 표시된 원이 나타납니다. 이 아이콘을 클릭하여 취소 프로세스를 시작하십시오.
