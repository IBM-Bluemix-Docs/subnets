# FAQ

## Come vengono assegnati gli indirizzi IP del server alla rete {{site.data.keyword.BluSoftlayer_notm}}?

Eseguiamo il provisioning dei server con un solo indirizzo IPv4 primario privato associato al dispositivo.  Ogni server può facoltativamente avere un solo indirizzo IPv4 primario pubblico e un solo indirizzo IPv6 primario pubblico.  Tutti gli altri indirizzi IP associati a un server devono essere assegnati da una sottorete secondaria, che può essere acquistata tramite il [Customer Portal ![Icona link esterno](../../icons/launch-glyph.svg "Icona link esterno")](https://control.softlayer.com/).

Le sottoreti sono create per i clienti caso per caso, che implica un processo accurato che cerca un rapporto da IP a server.
